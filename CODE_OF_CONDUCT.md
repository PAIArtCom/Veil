# Code of Conduct

Veil follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Standard

Project spaces should stay respectful, technical, and focused on improving the software.
Examples of expected behavior include:

- using welcoming and inclusive language;
- respecting different viewpoints and experience levels;
- accepting constructive feedback;
- keeping discussions focused on the issue, evidence, and tradeoffs.

Unacceptable behavior includes harassment, insults, threats, publishing private
information, or sharing secrets, credentials, customer data, or vulnerability details in
public project spaces.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, and other
contributions that violate this code. Repeated or severe violations may result in a ban
from project spaces.

Security issues must follow [SECURITY.md](SECURITY.md), not public discussion.
